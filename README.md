# 실전 Bash 쉘 프로그래밍 — 실습 코드

> 이 저장소는 **[실전 Bash 쉘 프로그래밍](https://text.ibetter.kr/bash)** 교재의 실습 코드를 담고 있습니다.

## 기술 스택

Bash 5.x, Ubuntu 24.04, ShellCheck

## 브랜치 구조

각 브랜치는 해당 챕터까지 누적된 전체 프로젝트 코드를 담고 있습니다.

| 브랜치 | 내용 |
|--------|------|
| `main` | 최종 완성 코드 |
| `part01/chapter-01` | PART 01 Ch.01까지의 코드 |
| `part01/chapter-02` | PART 01 Ch.02까지의 코드 |
| `partNN/chapter-MM` | ... |

원하는 챕터의 브랜치를 클론하면 해당 시점의 스크립트를 바로 실행할 수 있습니다.

```bash
git clone -b part03/chapter-02 https://github.com/sung2ne/textbook-bash.git
```
